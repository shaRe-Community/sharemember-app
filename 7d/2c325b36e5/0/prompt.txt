### `deploy <service> [env]`

Build, transfer, and restart a service:

1. **Build** the Docker image from the source repo
2. **Transfer** via `docker save | ssh ... docker load`
3. **Restart** via `docker compose up -d <service-env>`
4. **ALWAYS restart Traefik** after any container recreate: `docker restart coolify-proxy`
5. **Wait 5 seconds**, then verify the service started by checking logs
6. **Verify all routes** respond (200/302)
